Leeds Cycle
Patterns Analysis

Summary

Analysis of ride volumes 2018-23 across 11 recording sites based on 1.2 million data points & +80% daily completeness over a continuous 6 year period.

Key technologies

Python, Pandas, Tableau, R Studio & Google Sheets & AI tools (Claude) to accelerate workflow by creating python scripts to run locally in R Studio.

Key findings

  • Weekday peak hour is 5-6PM with 361K rides, 613% higher than the weekend peak hour 2-3PM with 51K rides.
  • Eight out of eleven recording sites have seen ride volume fully recover post COVID.
  • Weekday post COVID ride volume is up 3.7% whilst weekend is up 41.8% compared to pre COVID levels.

Python data analysis, pandas data manipulation, exploratory data analysis, data visualization, geospatial analysis, time series analysis, and data storytelling.

UK Festival
Analysis

Summary

A comprehensive analysis of 206 UK festivals in 2025, combining web scraped festival data with 30 years of historical weather patterns to provide insights and predict optimal festival experiences through interactive visualizations.

Key technologies

Screaming Frog, Google Sheets, Python (Pandas, NumPy, SciPy), Open-Meteo Weather API, R Studio, Tableau.

Key findings

  • Festival Analysis - Typical festival duration is 3 days, July hosts most festivals but August has highest capacity, South East England holds most festivals.
  • Weather Analysis - Driest festival is Cross the Tacks (3.3% chance of significant rainfall), Warmest festival is Jazz Cafe Festival (14.3C-22.4C temperature range), best Overall Weather is Tolpuddle Martyrs Festival and Rally.

Skills demonstrated

Web Scraping & Data Collection, Data Cleaning & Transformation, API Integration, Statistical Analysis, Geographic Data Processing, Data Visualization using Tableau.
